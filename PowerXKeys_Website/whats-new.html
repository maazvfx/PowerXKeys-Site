Quick Start Guide

Getting started is incredibly simple:

  1. Launch PowerX Keys.exe.
  2. Navigate to the Manage tab on the left sidebar.
  3. Click on any existing hotkey card, or use the menu to add a new customized shortcut.
  4. Ensure the master toggle in the top-left is green and says "ACTIVE".

What's New in Version 1.0

Welcome to the first official release of PowerX Keys! This application was built from the ground up to provide the fastest, most reliable hotkey automation experience for Windows professionals.

Core Engine & Performance

  • Zero Delay Execution: Custom native engine integration ensures key presses are registered and executed at the hardware level instantly.
  • Standalone Executable: Fully packaged into a single lightweight .exe file for immediate use.
  • Memory Optimized: Background engine uses virtually zero CPU and minimal RAM, even with complex setups.

Application Management

  • Per Browser Filters: Map hotkeys to trigger selectively across Chrome, Edge, Brave, and Firefox.
  • Global Overrides: Choose whether a hotkey runs everywhere or only in specific applications.
  • Portable Configuration: Settings are automatically isolated to the AppData layer for permanent persistence.

Professional Features

  • High Performance Media Controls: Native system commands for Volume (Esc + Up/Down) and advanced playback.
  • Smart Tab Switching: Optimized browser tab switching logic for maximum navigation speed.
  • Instant Capture UI: Assign keys by physically pressing them in the dashboard. No manual entry required.
